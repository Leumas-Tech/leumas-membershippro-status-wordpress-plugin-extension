A plugin to fetch user membership status using Paid Memberships Pro.

== Description ==

This plugin provides a custom REST API endpoint to fetch user membership status using Paid Memberships Pro.

== Installation ==

1. Upload the `pmpro-membership-status` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the endpoint `/wp-json/custom/v1/membership-status?username=<username>` to fetch user membership status.
